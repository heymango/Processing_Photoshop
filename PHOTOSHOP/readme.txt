 
1. 레이어
레이어는 총10개 이미지 레이어와 드로잉 레이어를 구분하였으며, 각각 최대 5개씩 생성하도록 하였다

레이어의 기본 옵션은 투명 레이어이다. 

-우측의 레이어박스 하단의 레이어 추가 아이콘을 누르면, 드로잉 레이어가 생성된다.
-현재 레이어(편집하고자 하는레이어)를 선택하면, 하늘색으로 하이라이트되며 표시가 된다.
-하이라이트된 레이어는 드로잉이 가능하며, 레이어칸에 눈 모양 아이콘을 누르면 레이어 숨기기가 가능하다

이미지레이어는 상단의 file 아이콘을 누르면 내 컴퓨터에서 원하는 이미지를 불러올 수 있다.
이미지 불러오기가 끝나면 자동적으로 이미지 레이어가 생성된다.

이미지 레이어는 필터링이 가능하다

2. 기능

  2.1 드로잉
      -드로잉 레이어에는 그림판처럼 그림 그리기가 가능하다.
      -드로잉 레이어의 기본 배경은 투명레이어이며, 좌측의 페인트 아이콘을 통해, 현재 피커에 선택되어있는
      색으로 레이어를 채울 수 있다.
      (페인트통을 클릭하여 on 되면,cusor가 바뀌고, 그 때 드로잉 레이어를 클릭하면 색이 입혀진다)
      -좌측의 드로잉 아이콘을 선택하면, 브러쉬 모드 변경, 크기조절, 지우개 를 선택할 수 있는 창이 열린다.
      -브러쉬는 원형과 사각형이 있으며, 크기 조절은 버튼으로써 가능하다. 현재 브러쉬의 크기를 실시간으로 보여줌으로서
      조절이 용이하다.
      -지우개는 단순히 배경색으로 칠하는 지우개가 아니라, 드래그한 부분의 해당 픽셀을 투명하게 만드는 진짜 지우개이다. 
      따라서 배경이 투명인 드로잉 레이어에서도 지우기가 가능하다.
     -좌측의 지우개 모양 기능은, clear all 기능으로써, 클릭하면 레이어의 모든 내용이 사라진다
     -드로잉 할때의 색상은, 현재 색상 피커에 선택된 색을 기반으로 그려진다(좌측의 컬러박스)
      좌측 메뉴의 최하단의 컬러 박스를 누르면, 색상을 선택할수 있는 컬러피커박스가 나온다 
      박스에서 클릭해 선택한 색상이, 현재 색상이 된다. 언제든지 변경가능하다 

.  2.2 이미지 프로세싱
     -이미지 레이어의 경우 drag&drop으로 레이어 위 이미지의 위치 조절이 가능하다.
      좌측의 화살표버튼을 누르면 이미지 조정 가능 상태가 된다.

    -상단의 사진필터 아이콘을 누르면, 이미지에 씌울수 있는 필터를 선택할 수 있는 box가 나타난다
     filter에는 THRESHOLD,GRAY,INVERT,POSTERIZE,BLUR,ERODE,DILATE,TINT 가 있으며, 
     THRESHOLD, POSTERIZE,BLUR 필터는 값을 드래깅하여 조절할 수 있다. TINT는 현재 피커에 있는 색상에 대한 TINT가 씌워진다

 2.3 불러오기&저장
   -사진 불러오기는 윈도우 탐색기가 열리면서 내 컴퓨터에서 직접 파일을 선택 가능하다
   -마찬가지로 사진 저장 또한 윈도우 탐색기가 열리면서 내가 원하는 디렉토리에 원하는 이름으로 저장가능하다
   -상단의 첫번째 저장 버튼(LAYER)은 현재 보고있는 레이어만 저장하는 것이고, 두번째 저장버튼(ALL)은
    쌓여진 레이어의 최종 모습, 즉 레이어 전체가 겹쳐진 이미지를 저장하는 기능이다.
